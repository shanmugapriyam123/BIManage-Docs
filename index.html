Installation

System Requirements

  • Autodesk Revit 2021, 2022, 2023, 2024, 2025, or 2026
  • Windows 10/11 (64-bit)
  • .NET Framework 4.8 (Revit 2021–2024) or .NET 8.0 (Revit 2025–2026)
  • Internet connection for API sync (offline mode available)

Install Steps

  1. Run ZeManageSetup.exe
  2. Accept the Privacy Policy
  3. Enter your license key (provided by your administrator)
  4. Select Revit versions to install for
  5. Complete installation and restart Revit

Protection

Pin Protection

Lock individual elements to prevent accidental modification. Protected elements cannot be moved, deleted, or modified without admin authorization.

Command Protection

Control which Revit commands can be executed on protected elements.

Supported CommandsProtection Modes
Delete, Cut, Copy, Move, Rotate, Mirror, Group, Unpin Notify Assist Protect
  • Notify — Log the action, allow user to proceed
  • Assist — Show guidance dialog, user decides
  • Protect — Block action entirely, require OTP or admin password

Event Restriction

Restrict Revit-level events based on company policies:

  • Open file from non-approved location
  • Open central file directly
  • Model upgrade protection
  • Save over earlier file version
  • CAD import/explode protection
  • Sync conflict detection
  • Family loading/version mismatch
  • Revit Link pin prompt

Rule-Based Protection

Define custom rules based on parameters, categories, or element types.

ScopeDescription
Company-wideApplied to all projects and models
Project-wideApplied to all models within a project
Model-specificApplied to a single model only

Admin Override

  • OTP (One-Time Password) — Generated by admin, single-use override code
  • Admin Password — Direct password entry for immediate override
  • Evidence Capture — Automatic screenshots before/after protected operations

Health Monitor

Element Breakdown

Comparisons

Levels vs Grids, Views vs Sheets, Worksets vs Families.

Sync Control

Background Sync

FeatureDescription
Continuous ModeSyncs on a fixed interval (default: 30 min)
When Paused ModeOnly syncs when user is idle (default: 15 min idle)
ScheduleRestrict sync to a time window (e.g., 21:00 – 09:00)
Compact ModelAuto-compact on first sync of each day
RelinquishAuto-release workset ownership when idle (default: 60 min)
Exit on IdleClose Revit after configurable idle period with countdown

Import / Export

Export and import sync settings as .ze files (JSON format) for sharing configurations across machines or projects.

  • Follow-up question suggestions for guided workflows

API Sync

All data syncs to the cloud via REST API with offline queue fallback:

  • HTTP REST — POST (create), PATCH (update), PUT (replace), DELETE
  • SignalR — Real-time push notifications for protection changes, chat, presence
  • Offline Queue — Failed operations queued locally, auto-retried when connection restored
  • Device Auth — License key registration, token-based authentication

Settings

Sync Settings

Configure background sync, relinquish, scheduling, compact model, and exit-on-idle from the Sync Settings dialog.

Model Registration

Register models for tracking. Each model gets a unique GUID for identification across sessions.

OTP Management

Administrators can generate one-time passwords for users who need to override protection temporarily.
